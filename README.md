# Proyecto Legion

Bienvenido al proyecto **Legion**, un servidor de hosting moderno y minimalista. Este proyecto está diseñado para proporcionar una experiencia de usuario sencilla y efectiva, utilizando una paleta de colores en negro, violeta y gris.

## Estructura del Proyecto

El proyecto tiene la siguiente estructura de archivos:

```
legion-landing
├── public
│   ├── index.html        # Documento HTML principal para la página de aterrizaje
│   ├── css
│   │   └── styles.css    # Estilos para la página de aterrizaje
│   └── js
│       └── main.js       # Código JavaScript para elementos interactivos
├── assets
│   └── legion-icon.png    # Logo de Legion
├── .gitignore             # Archivos y directorios a ignorar por Git
├── package.json           # Configuración de npm
└── README.md              # Documentación del proyecto
```

## Instalación

Para comenzar a trabajar con el proyecto, sigue estos pasos:

1. Clona el repositorio:
   ```
   git clone <URL_DEL_REPOSITORIO>
   ```
2. Navega al directorio del proyecto:
   ```
   cd legion-landing
   ```
3. Instala las dependencias:
   ```
   npm install
   ```

## Uso

Para iniciar el proyecto, abre el archivo `public/index.html` en tu navegador. La página mostrará el logo de Legion y un mensaje indicando que el servidor no está disponible en este momento.
